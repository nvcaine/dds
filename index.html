Additional details:

This sections aims to explain in details different aspects of the application.

First, let's take at a look at the tools used:

  1. jQuery 1.11

  2. Twitter Bootstrap 3.1

    Cross-browser responsive layout library.

  3. Handlebars 1.3

    JavaScript-based template engine.

  4. jQuery Pep 0.6

    Cross-platform drag/drop event wrapper.

    The module offers mainly dragging and only some rudimentary dropping. This module was used mainly to bridge the gap between desktop and mobile devices. Specifically, to provide a wrapper for mouse/touch events.

    Added functionality: droppable targets dispatch 'dragover', 'dragout' and 'dropover' events.
